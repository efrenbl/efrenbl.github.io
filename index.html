Glosario

Algoritmos Evolutivos
Métodos de optimización inspirados en la evolución biológica que trabajan con una población de soluciones, seleccionándolas, combinándolas y mutándolas para mejorar su calidad.
Lógica Difusa
Extensión de la lógica clásica que maneja conceptos imprecisos mediante grados de verdad, permitiendo decisiones en ambientes con información ambigua o incierta.
Agentes
Entidades que perciben y actúan en un entorno, basándose en reglas o algoritmos, para maximizar algún beneficio o recompensa.
Sistemas Expertos
Programas que imitan el razonamiento de expertos humanos en áreas específicas, basándose en un conjunto de reglas y un banco de conocimientos.
Aprendizaje Supervisado
Tipo de aprendizaje en el que el modelo se entrena con un conjunto de datos etiquetado, es decir, cada muestra de entrenamiento tiene un resultado o etiqueta asociada.
Aprendizaje No Supervisado
Tipo de aprendizaje en el que el modelo se entrena con un conjunto de datos sin etiquetar, buscando encontrar patrones y estructuras en los datos.
Aprendizaje por Reforzamiento
Tipo de aprendizaje donde el modelo aprende a tomar decisiones a través de la interacción con un entorno, recibiendo recompensas o penalizaciones.
Algoritmo de Aprendizaje
Un procedimiento o fórmula que se utiliza para adaptar un modelo a los datos.
Modelo
Una representación matemática o computacional basada en los datos de entrenamiento, que se utiliza para hacer predicciones o decisiones.
Características
Las variables o atributos del conjunto de datos que se utilizan para entrenar el modelo.
Etiquetas
Los resultados o valores objetivo en un conjunto de datos etiquetado, utilizados en aprendizaje supervisado.
Conjunto de Entrenamiento
Subconjunto de datos utilizado para entrenar el modelo.
Conjunto de Pruebas
Subconjunto de datos utilizado para evaluar el rendimiento del modelo.
Validación Cruzada
Técnica de evaluación del modelo que divide el conjunto de datos en varios subconjuntos, entrena el modelo en algunos de ellos y lo evalúa en los restantes.
Sobreajuste
Cuando un modelo aprende demasiado bien los datos de entrenamiento, incluyendo el ruido, y generaliza mal a nuevos datos.
Subajuste
Cuando un modelo no aprende suficientemente bien la estructura de los datos de entrenamiento y generaliza mal a nuevos datos.
Regresión
Tipo de tarea de aprendizaje supervisado que predice un valor continuo.
Clasificación
Tipo de tarea de aprendizaje supervisado que predice una etiqueta categórica.
Clustering
Tarea de aprendizaje no supervisado que agrupa datos similares.
Reducción de Dimensionalidad
Técnica que reduce el número de características mientras conserva la estructura de los datos.
Hiperparámetros
Parámetros del algoritmo de aprendizaje que se deben configurar antes del entrenamiento y no se aprenden de los datos.
Métricas de Evaluación
Medidas utilizadas para evaluar el rendimiento de un modelo, como la precisión, la exhaustividad y el área bajo la curva ROC.
Grid Search
Técnica para seleccionar los hiperparámetros óptimos de un modelo mediante la exploración exhaustiva de un conjunto predefinido de valores de hiperparámetros.
Scikit-Learn (sklearn)
Biblioteca de Python para machine learning que proporciona herramientas simples y eficientes para el análisis de datos.
ID3 (Iterative Dichotomiser 3)
Es uno de los primeros algoritmos desarrollados para generar árboles de decisión. Fue propuesto por Ross Quinlan en 1986. El algoritmo utiliza un enfoque top-down y emplea el concepto de entropía y ganancia de información para elegir el atributo que mejor divide el conjunto de datos en cada nivel del árbol.
C4.5
Es una extensión y mejora del algoritmo ID3, también desarrollado por Ross Quinlan. C4.5 introdujo varias mejoras, como el manejo de atributos con valores faltantes y la capacidad de trabajar con atributos tanto discretos como continuos. Además, en lugar de utilizar solamente la ganancia de información, C4.5 emplea una métrica llamada "ratio de ganancia" para evitar un sesgo hacia atributos con muchos valores distintos..
Boosting
Es una estrategia que entrena modelos de manera secuencial, donde cada nuevo modelo se enfoca en corregir los errores cometidos por los modelos anteriores. A través de esta corrección iterativa, el boosting busca incrementar la precisión del modelo final, reduciendo tanto el sesgo como la varianza. Algunos algoritmos populares basados en boosting incluyen AdaBoost, Gradient Boosting Machines (GBM) y XGBoost.
Bagging (Bootstrap Aggregating)
Es una técnica de ensemble learning que implica entrenar varios modelos en paralelo utilizando subconjuntos aleatorios de los datos de entrenamiento, creados mediante muestreo con reemplazo. El objetivo principal de bagging es reducir la varianza del modelo final y evitar el sobreajuste, siendo el algoritmo de bosques aleatorios (Random Forest) una aplicación destacada de esta técnica.
Deep Learning (Aprendizaje Profundo)
Subárea del machine learning basada en redes neuronales profundas capaces de aprender patrones complejos directamente de grandes cantidades de datos, automatizando la extracción de características y siendo eficaz en tareas como reconocimiento de imágenes y procesamiento de lenguaje natural.
BERT (Bidirectional Encoder Representations from Transformers)
Modelo de procesamiento de lenguaje natural que utiliza transformers para entender el contexto de las palabras en un texto, analizando simultáneamente las palabras anteriores y posteriores.
GPT (Generative Pre-trained Transformer)
Modelo de lenguaje basado en transformers entrenado primero en grandes cantidades de texto, conocido por generar texto coherente y fluido y adaptarse a tareas específicas.
ResNet (Residual Networks)
Arquitectura de red neuronal diseñada para el reconocimiento de imágenes que introduce "conexiones residuales", facilitando el entrenamiento de redes con muchas capas.
Ecuación polinómica
es una expresión matemática que representa una relación entre variables utilizando sumas de términos, donde cada término es el producto de un coeficiente y una variable elevada a una potencia entera no negativa.
